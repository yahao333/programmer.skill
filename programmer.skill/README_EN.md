# Programmer.skill

Distill a programmer's coding style, problem-solving approach, and tech wisdom into an AI Skill.

## Usage

| Command | Description |
|---------|-------------|
| `/programmer` | Invoke the Programmer Skill creator |

## Description

Programmer.skill is a Claude Code extension that distills a programmer's coding style, problem-solving approach, and tech wisdom into an AI-powered skill.

### Workflow

1. **Input Info**: Name, basic info, tech style
2. **Import Materials**: Code snippets, CR records, chat records, etc.
3. **Analyze & Generate**: Extract coding style, problem-solving approach, communication traits
4. **Confirm & Write**: Generate Programmer Skill file

### Material Types

- Code snippets
- Code Review records
- Technical docs/blogs
- Chat records
- Paste text directly
